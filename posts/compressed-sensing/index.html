A Quick Formulation of Compressed Sensing

2021-12-27 | A very, very mininal derivation of the optimization

Compressed sensing is a singal processing method for reconstructing a signal from an undersampled set of measurements. Here, we will derive the optimization problem at the core of compressed sensing.

  • Let xxx be the true signal.

  • Let x↦Usx \stackrel{\mathcal U}{\mapsto} sx↦U​s be the representation of xxx in a functional basis space.

  • Let x=Ψsx = \Psi sx=Ψs for some universal transform basis Ψ\PsiΨ (e.g., Fourier, wavelet, etc.).

  • Let y=Cxy = Cxy=Cx be the measured signal, where CCC is the measurement matrix.

Our objective is to obtain a subsampled version of sss from yyy, and then recover s↦U−1xs \stackrel{\mathcal U^{-1}}{\mapsto} xs↦U−1​x, i.e.,

min⁡s12∣∣y−CΨs∣∣22 . \min_s \frac{1}{2} ||y - C\Psi s||_2^2 \,. smin​21​∣∣y−CΨs∣∣22​.

As stated, this optimization is a underspecified / indeterminate problem (i.e., there are infinitely many solutions for sss), so we constrain the solutions to be sparse:

min⁡s12∣∣y−CΨs∣∣22+λ∣∣s∣∣0 . \min_s \frac{1}{2} ||y - C\Psi s||_2^2 + \lambda||s||_0 \,. smin​21​∣∣y−CΨs∣∣22​+λ∣∣s∣∣0​.

However, minimizing l0l_0l0​ norms is NP hard,[1] so we relax the problem with the l1l_1l1​ norm:

min⁡s12∣∣y−CΨs∣∣22+λ∣∣s∣∣1 . \min_s \frac{1}{2} ||y - C\Psi s||_2^2 + \lambda||s||_1 \,. smin​21​∣∣y−CΨs∣∣22​+λ∣∣s∣∣1​.

[1] Because it requires combinatorial optimization.
This is a well-defined problem, and is solvable with linear programming (e.g., basis pursuit, denoising/matching). Additionally, Equation 3 is equivalent to Equation 2 under certain conditions (e.g., ν\nuν-incoherence) and constraints on CΨC\PsiCΨ. Then, given a noisy, downsampled measurement yyy, solve Equation 3 to get s^\hat ss^, and predict the original signal to be x^=Ψs^\hat x = \Psi \hat sx^=Ψs^ which is equivalent to s^↦U−1x^\hat s \stackrel{\mathcal U^{-1}}{\mapsto} \hat xs^↦U−1​x^.
